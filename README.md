# CleanBase Generator

CleanBase Generator is a monorepo for a production-ready MVP that generates standardized .NET and Angular starter solutions using validated configuration, opinionated architecture rules, and template-first generation.

## Current Phase

- ✅ Phase 1 completed (requirements + architecture alignment)
- ✅ Phase 2 completed (backend foundation)
- ⏳ Phase 3 pending (Angular UI)

## Repository Layout

- `src/backend`: .NET backend for validation, preview, generation orchestration, and job tracking.
- `docs`: phase notes and implementation decisions.
